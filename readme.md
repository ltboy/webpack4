# 一个简单的 `vue-cli@2` `template模板`

> 参考项目[vuejs-templates/browserify-simple](https://github.com/vuejs-templates/browserify-simple)

> **注意事项:** 
1. 本项目为个人需求定制 不适用于复杂需求
2. webpack采用的是4.x版本

### 使用方法

```bash
$ npm install -g vue-cli
$ vue init ltboy/vue-template@2 my-project
$ cd my-project
$ yarn
$ yarn dev
```
